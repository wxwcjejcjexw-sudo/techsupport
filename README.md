TechSupport Pro

Локальный инструмент для обслуживания Windows, который работает прямо из коробки. Без установки Python, Node.js и прочих зависимостей. Просто скачал, запустил и пользуешься.

Что умеет

Внутри собрано больше 40 команд, разбитых на шесть категорий:

Очистка — временные файлы, кэш браузеров, корзина, Prefetch, старые обновления Windows. Всё, что засоряет диск.

Сеть — сброс DNS, обновление IP, диагностика соединения, сброс Winsock. Помогает когда интернет работает через раз.

Система — информация о железе, проверка диска, сканирование системных файлов, дефрагментация, список процессов.

Безопасность — статус брандмауэра, Windows Defender, открытые порты, аудит настроек.

Диагностика — здоровье диска, отчёт о батарее, просмотр событий, драйверы, проверка памяти.

Инструменты — быстрый доступ к диспетчеру задач, устройствам, панели управления, реестру, PowerShell.

Плюс три комплексных сценария: полная диагностика, быстрое исправление сети и оптимизация системы.

Как запустить

Простой способ: дважды кликни по start.bat. Откроется консоль, стартует локальный сервер на порту 8080, и браузер сам откроется с интерфейсом.

Ручной способ: если хочешь контролировать процесс, запусти server.ps1 через PowerShell с правами обычного пользователя.

Требования

Windows 10 или 11. PowerShell 5.1 или новее (есть в системе по умолчанию). Права администратора нужны только для некоторых команд — приложение само запросит повышение, когда потребуется.

Как это работает

Приложение состоит из трёх частей. PowerShell-сервер слушает локальный порт и выполняет системные команды. HTML-интерфейс отображается в браузере и отправляет запросы на сервер. JavaScript обрабатывает взаимодействие, анимации и вывод результатов.

Всё работает локально. Никаких облаков, никакой отправки данных наружу. Даже интернет не нужен после первой загрузки шрифтов.

Особенности

Интерфейс адаптирован под русский язык, но кодировка UTF-8 корректно обрабатывается во всех скриптах. Прогресс-бары и анимации делают процесс наглядным. Результаты команд можно копировать одной кнопкой. Системная информация обновляется в реальном времени.

Безопасность

Все команды — стандартные системные утилиты Windows. Никаких сторонних бинарников, никакой скрытой активности. Исходный код открыт, можешь проверить каждую строчку в server.ps1.

Если что-то пошло не так

Порт 8080 занят — закрой другие приложения или измени порт в server.ps1, строка с $Port.

PowerShell не найден — странно для современной Windows, но можешь скачать с сайта Microsoft.

Команды требуют прав администратора — это нормально для системных операций, приложение само поднимет окно UAC.

Браузер не открылся автоматически — перейди вручную на localhost:8080.
