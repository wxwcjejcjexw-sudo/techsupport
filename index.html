Команды

Выберите категорию и выполните нужную команду

Безопасно

Очистка папки TEMP

Удаляет временные файлы пользователя из папки %TEMP%. Освобождает место на диске.

Что делает: Удаляет временные файлы, созданные программами
Риски: Минимальные. Файлы можно удалить безопасно
Требует админа

Очистка Windows Temp

Удаляет системные временные файлы из C:\Windows\Temp

Что делает: Очищает системную папку временных файлов
Риски: Требует права администратора
Требует админа

Очистка Prefetch

Удаляет файлы предзагрузки. Может немного замедлить первый запуск программ.

Что делает: Очищает кэш предварительной загрузки
Риски: Первый запуск программ может быть медленнее
Безопасно

Очистка корзины

Полностью очищает корзину. Удалённые файлы нельзя будет восстановить.

Что делает: Безвозвратно удаляет файлы из корзины
Риски: Файлы нельзя восстановить после очистки
Безопасно

Очистка кэша браузеров

Очищает кэш Chrome, Edge, Firefox. Может потребоваться повторный вход на сайты.

Что делает: Удаляет временные файлы браузеров
Риски: Возможен выход с сайтов, сохранённые пароли не затрагиваются
Требует админа

Очистка обновлений Windows

Удаляет старые файлы обновлений Windows. Освобождает значительное место.

Что делает: Удаляет кэш Windows Update
Риски: Нельзя будет откатить обновления
Безопасно

Сброс DNS кэша

Очищает кэш DNS. Помогает при проблемах с открытием сайтов.

Что делает: Очищает локальный кэш DNS-резолвера
Когда использовать: Сайт не открывается, но интернет работает
Безопасно

Обновление IP-адреса

Освобождает и получает новый IP от DHCP сервера. Кратковременно прервёт соединение.

Что делает: ipconfig /release + ipconfig /renew
Когда использовать: IP-конфликт, нет интернета по DHCP
Требует админа

Полный сброс сети

Сбрасывает Winsock и IP-стек. Требуется перезагрузка после выполнения.

Что делает: netsh winsock reset + netsh int ip reset
Риски: Обязательна перезагрузка после выполнения
Только чтение

Информация о сети

Показывает IP, MAC, шлюз для активных сетевых адаптеров.

Что делает: Выводит информацию о сетевых адаптерах
Риски: Нет. Только просмотр информации
Только чтение

Проверка соединения

Пингует Google DNS (8.8.8.8). Проверяет наличие интернет-соединения.

Что делает: ping 8.8.8.8 -n 4
Риски: Нет. Только диагностика
Требует админа

Что делает: netsh winsock reset
Риски: Рекомендуется перезагрузка
Только чтение

Проверяет диск C: на ошибки. Может потребоваться перезагрузка.

Что делает: chkdsk C: /F
Риски: Может запуститься при перезагрузке
Требует админа

Что делает: sfc /scannow
Время: 5-15 минут
Безопасно
